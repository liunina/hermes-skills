# Hermes Skills

Collection of skills for [Hermes Agent](https://hermes-agent.nousresearch.com/).

## Available Skills

### wikijs
Deploy and manage Wiki.js v2 — Docker deployment with PostgreSQL, nginx reverse proxy, locale configuration, GraphQL API CRUD, and programmatic page management.

## Installation

```bash
# Add this repo as a skill source
hermes skills tap add https://github.com/liunina/hermes-skills

# Install a skill
hermes skills install wikijs
```

## Update

```bash
hermes skills update
```

Or pull the latest from the repo and re-install.
